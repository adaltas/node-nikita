
# `nikita.docker.login`

Register or log in to a Docker registry server.

## Output

* `err`   
  Error object if any.   
* `$status`   
  True when the command was executed successfully.
* `stdout`   
  Stdout value(s) unless `stdout` option is provided.
* `stderr`   
  Stderr value(s) unless `stderr` option is provided.
